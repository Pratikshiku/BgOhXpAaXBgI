### 一、相关技术
- 后端：`Java`、`JavaWeb` / `Springboot`。
- 前端：`Vue`、`HTML` / `CSS` / `Javascript` 等。
- 数据库：`MySQL`

### 二、相关软件（列出的软件其一均可运行）
- `IDEA`
- `Eclipse`
- `Visual Studio Code(VScode)`
- `Navicat`
- 等

### 三、功能描述
系统分为（身份）：普通用户、员工、管理员。

**普通用户功能：**
1. 登录
2. 注册
3. 首页
4. 菜品
5. 论坛
6. 公告信息
7. 酒店介绍
8. 联系我们
9. 个人中心
10. 菜品收藏
11. 菜品订单
12. 后台管理



**员工功能：**
1. 登录
2. 注册
3. 修改密码
4. 个人信息
5. 菜品管理、菜品评价管理、菜品收藏管理、菜品订单管理
6. 论坛管理
7. 供应商管理
8. 公告信息管理



**管理员功能：**
1. 登录
2. 个人中心（修改密码、个人信息）
3. 管理员管理
4. 菜品管理、菜品评价管理、菜品收藏管理、菜品订单管
5. 基础数据管理（菜品类型管理、物品类型管理、会员等级类型管理、公告类型管理、单页数据类型管理）
6. 论坛管理
7. 供应商管理
8. 公告信息管理
9. 单页数据管理
10. 用户管理
11. 员工管理
12. 轮播图信息
